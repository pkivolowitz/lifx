How will you use GlowUp?

GlowUp can run as a simple command-line tool or as a full server with a dashboard, scheduler, and optional voice control. Choose the mode that fits your setup.

CLI-only mode gives you

  • Run lighting effects directly from the terminal
  • Full effect library: color cycles, pulses, flames, gradients, audio-reactive, and more
  • Works on any computer on the same network as your LIFX bulbs
  • No server, no background services, no configuration files
  • Minimal install: just the core engine and LIFX transport

CLI-only mode does not include

  • Web dashboard for controlling lights from a browser
  • Scheduler for time-of-day and sunrise/sunset automation
  • Always-on server — effects stop when you close the terminal
  • Sensor integration (Zigbee, BLE, motion, soil moisture)
  • Voice control
  • Kiosk displays

Choosing “No” opens up

  • An always-on server with web dashboard and REST API
  • Scheduled effects tied to time of day, sunrise, and sunset
  • Optional sensor adapters: Zigbee plugs, BLE beacons, Matter devices
  • Optional voice control with wake-word detection
  • Multi-room voice satellites and kiosk displays
  • Distributed deployment across multiple machines
