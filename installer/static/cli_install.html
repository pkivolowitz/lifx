CLI-Only Installation

This will install the GlowUp command-line effect engine. Effects run directly from your terminal against LIFX bulbs on your local network.

GlowUp files will be placed here. The directory will be created if it does not exist.
Isolates GlowUp from your system Python. The venv will be created inside the installation directory. The core CLI has no pip dependencies, but a venv keeps things clean for future upgrades.
